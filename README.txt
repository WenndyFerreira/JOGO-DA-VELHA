# Jogo da Velha Noir 👻

Este é um projeto de **Jogo da Velha** desenvolvido com foco em aprendizado e prática de lógica de programação. O projeto conta com duas versões: uma para desktop (Python) e uma para web (HTML/JS).

## 🚀 Sobre o Projeto
O jogo foi criado como um exercício prático para o curso de **Análise e Desenvolvimento de Sistemas (ADS)**. O objetivo principal foi aplicar conceitos de:
- Lógica de estados e condições de vitória.
- Manipulação de interface gráfica (GUI) e DOM.
- Estilização personalizada (minimalismo preto, branco e roxo).
- Transição entre múltiplas interfaces (Login e Game).

## 🎨 Visual
O projeto segue uma estética "Noir", utilizando uma paleta de cores restrita para um visual moderno e limpo:
- **Fundo:** Preto
- **Elementos:** Branco
- **Nicks/Status:** Roxo (#A020F0)

## 🛠️ Tecnologias Utilizadas
- **Python** (Biblioteca Tkinter)
- **HTML5**
- **CSS3**
- **JavaScript**

## 📖 Como utilizar
Sinta-se à vontade para clonar este repositório, estudar o código ou sugerir melhorias. Este projeto é de código aberto para fins educacionais.
